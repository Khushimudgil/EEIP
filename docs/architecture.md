# Enterprise Engineering Intelligence Platform (EEIP)

## Goal

Build an AI-powered platform that helps engineering teams understand repositories, perform semantic code search, generate documentation, review code quality, and assist developers through specialized AI agents.

## Core Features

- Repository Upload
- Repository Understanding
- Semantic Search
- RAG Pipeline
- Architecture Analysis
- Documentation Generation
- Code Review
- Test Case Generation
- Multi-Agent Workflows

## Tech Stack

### Backend
- FastAPI

### Frontend
- Next.js

### Vector Database
- ChromaDB

### LLM
- Gemini

### Agent Framework
- LangGraph

### Database
- SQLite (initially)
